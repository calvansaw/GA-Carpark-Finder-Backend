| ------- | --------- | ---------- | ------------- | --------------- |

|         |           | / comment  | GET           | Read comments   |
|         |           | / register | GET           |                 |
| 2.      | Create    | / carpark  | POST          |                 |
|         |           | / comment  | POST          | create comments |
|         |           | / register | POST          | Register form   |
|         |           | / login    | POST          | Login form      |
